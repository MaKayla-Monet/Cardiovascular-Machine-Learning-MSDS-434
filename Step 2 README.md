## Step 2 - Identify the Dataset

### Dataset Overview 
This project uses the Cardiovascular Disease dataset from Kaggle. 
Dataset Link: https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset

### Key Featrues
- Age
- Height
- Weight
- Gender
- Systolic blood pressure
- Diastolic blood pressure
- Cholestero
- Glucose
- Smoking
- Alcohol intake 
- Physical activity 
- Presence or absence of cardiovascular disease 

### Github Purpose
Store the project repository, documentation, and progress updates throughout the project development process.
